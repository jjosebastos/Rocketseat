# Tipos de dados

Conforme o ECMAScript (padroniza o JS) standard temos 9 tipos de dados:

* Data types
    * Primite / Primitive value
    * Structural
    * Structural Primite

## Primitivos

* String 
* Number 
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais 

* Object
    * Array 
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Structural Root Primite

* null
